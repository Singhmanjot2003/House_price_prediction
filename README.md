In this project, I developed a machine learning model to predict house prices based on various features such as location factors, size, and other property-related variables. I performed data cleaning and preprocessing, including handling missing values and checking for outliers, followed by exploratory data analysis (EDA) to understand relationships between features. Using Python libraries like Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn, I built and trained a Linear Regression model and evaluated its performance using statistical metrics. The results showed that the model was able to predict house prices with reasonable accuracy, and key features significantly influenced the price, demonstrating the effectiveness of regression techniques in real estate price estimation.
